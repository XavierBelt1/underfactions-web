📁 Complementos del Servidor
[ X ] CERRAR

Para vivir la experiencia completa de UnderFactions — escuchar música, voz de proximidad y más — necesitas instalar algunos complementos en tu Minecraft. Sigue estos pasos:

Paso 1 — Instala Fabric

Fabric es el cargador de mods que necesitas. Si usas TLauncher o SKLauncher ya lo puedes instalar desde el launcher.

1. Descarga Fabric desde → fabricmc.net
2. Selecciona versión 1.21.4
3. Instala y abre Minecraft con el perfil Fabric
Paso 2 — Descarga PlasmoVoice

PlasmoVoice es el mod principal de música y voz del servidor. Sin él no escucharás los discos ni el audio de otros jugadores.

1. Ve a → modrinth.com/mod/plasmo-voice
2. Descarga la versión para Fabric 1.21.4
3. Copia el .jar a tu carpeta .minecraft/mods/
Paso 3 — Fabric API (requerido)

Fabric API es una dependencia obligatoria para que los mods funcionen correctamente.

1. Ve a → modrinth.com/mod/fabric-api
2. Descarga versión 1.21.4
3. Copia a tu carpeta .minecraft/mods/
¿Sin cuenta premium? — TLauncher

Si no tienes cuenta premium de Minecraft puedes usar TLauncher o SKLauncher con soporte para Fabric.

1. Descarga TLauncher → tlauncher.org
2. En el launcher selecciona perfil Fabric 1.21.4
3. Instala los mods en la carpeta mods de TLauncher
